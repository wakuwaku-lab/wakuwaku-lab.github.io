SFDC AI Inspect とは

既存の複数のツールを1つの洗練されたUIに統合した、モダンなChrome拡張機能。AI Agentを中心に、日常の運用・開発・調査を「会話ベース」で高速化できます。

8つのコア機能

🤖 AI Agent Workspace

Salesforceを自然言語で操作できるAIワークスペース

  • 質問すると、そのまま実務で使える回答を返す
  • 75+ のLLMプロバイダーに接続可能（GPT / Claude / Gemini）
  • コスト・性能に合わせてモデル/プロバイダーを切り替え可能
  • Local Model Ready（Ollama対応）
  • Custom MCP Integrationで機能を拡張可能

📊 Data Expert

データ管理をAIで加速する、Excelライクな実務UI

📋

Excel-like UI

見やすくレスポンシブなテーブルでデータを検索・編集

⚡

Bulk Operations

Insert / Update / Upsert / Delete の一括操作

🔍

SOQL Helper

フォーマットとフィールド補助でSOQL作成を高速化

🤖

AI to SOQL

自然言語からSOQLを生成

📜 Log Expert + AI Analysis

ログ調査をAIで短時間化し、原因特定を支援

  • 最新デバッグログをすぐ取得・確認
  • ユーザー、ステータス、時間など多軸で絞り込み
  • AIが根本原因の候補を要約

AI Summary

原因の候補、証拠、改善案をAIが自動要約

👔 企業導入のメリット

🚀 生産性(ROI)の向上

「隠れた項目」を探したり複雑なSOQLツールを立ち上げる時間を削減し、コアなビジネスロジックの開発に集中できます。

👁️ ナレッジの可視化

ER図やアーキテクチャマップを簡単に生成でき、属人化を防ぎ、新メンバーのキャッチアップを加速させます。

💰 コスト削減

複数のアドオンを使い分ける必要がなくなり、学習コストとツール管理の手間を最小限に抑えられます。

🔒 セキュリティとプライバシー

  • ローカル動作 - すべての処理はブラウザ内で完結
  • 外部送信なし - データや認証情報が外部サーバーに保存されることはなし
  • 直接通信 - 認証された自身のSalesforce組織とのみ通信
  • 標準API - Salesforceの標準APIとセーフなセッション管理を使用
