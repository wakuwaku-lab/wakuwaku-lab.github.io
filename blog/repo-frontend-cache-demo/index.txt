# フロントエンドキャッシュデモ

Webアプリケーションにおける6つのキャッシュレベルをインタラクティブに学ぶデモアプリケーションです。

## キャッシュレベル

1. **コンポーネントキャッシュ** - React.memo、useMemo、useCallbackによるUI再レンダリングの最適化
2. **ランタイムメモリキャッシュ** - APIレスポンスや計算結果をRAMに一時保存
3. **永続キャッシュ** - localStorage/IndexedDBによるデータ永続化
4. **Service Workerキャッシュ** - プロキシーレイヤーによるネットワークリクエスト制御
5. **HTTPキャッシュ** - Cache-Control、ETagによるブラウザ組み込みキャッシュ
6. **CDNエッジキャッシュ** - 世界に分布するCDNサーバーによるキャッシュ

## 実行方法

### 簡易サーバーを使う

```bash
cd frontend-cache-demo
python3 -m http.server 8080
```

ブラウザで http://localhost:8080 にアクセスしてください。

### Dockerを使う

```bash
cd frontend-cache-demo
docker-compose up
```

ブラウザで http://localhost:8080 にアクセスしてください。

## 機能

- 各キャッシュレベルのインタラクティブなテスト
- 3言語対応（日本語、英语、中文）
- デフォルト言語：日本語
- キャッシュ動作の可視化

## 技術スタック

- Vanilla JavaScript
- HTML5/CSS3
- Service Worker API
- localStorage API

# Frontend Cache Demo

An interactive demo application to learn about 6 levels of caching in web applications.

## Cache Levels

1. **Component Cache** - UI re-render optimization using React.memo, useMemo, useCallback
2. **Runtime Memory Cache** - Temporarily store API responses and computed results in RAM
3. **Persistent Cache** - Data persistence using localStorage/IndexedDB
4. **Service Worker Cache** - Network request control via proxy layer
5. **HTTP Cache** - Browser built-in caching via Cache-Control, ETag
6. **CDN Edge Cache** - Cache distributed across worldwide CDN servers

## How to Run

### Using Simple Server

```bash
cd frontend-cache-demo
python3 -m http.server 8080
```

Open http://localhost:8080 in your browser.

### Using Docker

```bash
cd frontend-cache-demo
docker-compose up
```

Open http://localhost:8080 in your browser.

## Features

- Interactive testing of each cache level
- Support for 3 languages (Japanese, English, Chinese)
- Default language: Japanese
- Visual cache operation display

## Tech Stack

- Vanilla JavaScript
- HTML5/CSS3
- Service Worker API
- localStorage API
