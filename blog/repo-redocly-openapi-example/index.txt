# Redocly + OpenAPI サンプル

このプロジェクトでは、RedoclyとOpenAPI仕様を使用して美しいAPIドキュメントを生成する方法を説明します。

## 機能

- OpenAPI 3.0仕様によるAPI定義
- Redoclyによるドキュメント生成
- Express.jsサーバーと統合されたドキュメント
- ボリュームマウントを使用したDocker Compose設定

## クイックスタート

### Docker Composeを使用（推奨）

```bash
# APIサーバーとドキュメントサーバーを起動
docker-compose up

# APIサーバーのみ起動
docker-compose up api

# ドキュメントサーバーのみ起動
docker-compose up docs
```

ドキュメントへのアクセス: http://localhost:8080

### ローカル開発

```bash
# 依存関係をインストール
npm install

# APIサーバーを起動
npm run dev

# ドキュメントを生成
npm run docs:build

# ドキュメントをプレビュー
npm run docs:serve

# OpenAPI仕様をリント
npm run docs:lint
```

## エンドポイント

| エンドポイント | 説明 |
|----------------|------|
| `http://localhost:3000/api/v1/users` | ユーザーAPI |
| `http://localhost:3000/api/v1/products` | 商品API |
| `http://localhost:3000/api-docs` | Redocドキュメント |
| `http://localhost:3000/openapi.yaml` | OpenAPI仕様ファイル |
| `http://localhost:8080/redoc-static.html` | 静的ドキュメント（ビルド後） |

## プロジェクト構造

```
redocly-openapi-example/
├── api/
│   └── openapi.yaml      # OpenAPI仕様
├── src/
│   └── index.ts          # Expressサーバー
├── docs/                  # 生成されたドキュメント
├── docker-compose.yml
├── package.json
├── tsconfig.json
└── redocly.yaml          # Redocly設定
```

## コマンド

| コマンド | 説明 |
|----------|------|
| `npm run dev` | 開発サーバーを起動 |
| `npm run docs:lint` | OpenAPI仕様を検証 |
| `npm run docs:build` | 静的HTMLドキュメントを生成 |
| `npm run docs:serve` | ドキュメントをプレビュー |
| `npm run docs:bundle` | OpenAPI仕様をバンドル |

# Redocly + OpenAPI Example

This project demonstrates how to generate beautiful API documentation using Redocly and OpenAPI specification.

## Features

- OpenAPI 3.0 specification for API definition
- Redocly for documentation generation
- Express.js server with integrated docs
- Docker Compose setup with volume mounts

## Quick Start

### Using Docker Compose (Recommended)

```bash
# Start both API server and documentation server
docker-compose up

# Start only the API server
docker-compose up api

# Start only the documentation server
docker-compose up docs
```

Access the documentation at: http://localhost:8080

### Local Development

```bash
# Install dependencies
npm install

# Run the API server
npm run dev

# Generate documentation
npm run docs:build

# Preview documentation
npm run docs:serve

# Lint OpenAPI spec
npm run docs:lint
```

## Endpoints

| Endpoint | Description |
|----------|-------------|
| `http://localhost:3000/api/v1/users` | User API |
| `http://localhost:3000/api/v1/products` | Product API |
| `http://localhost:3000/api-docs` | Redoc documentation |
| `http://localhost:3000/openapi.yaml` | OpenAPI spec file |
| `http://localhost:8080/redoc-static.html` | Static docs (after build) |

## Project Structure

```
redocly-openapi-example/
├── api/
│   └── openapi.yaml      # OpenAPI specification
├── src/
│   └── index.ts          # Express server
├── docs/                  # Generated documentation
├── docker-compose.yml
├── package.json
├── tsconfig.json
└── redocly.yaml          # Redocly configuration
```

## Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run docs:lint` | Validate OpenAPI spec |
| `npm run docs:build` | Generate static HTML docs |
| `npm run docs:serve` | Preview documentation |
| `npm run docs:bundle` | Bundle OpenAPI spec |
