Redocly + OpenAPI サンプル

このプロジェクトでは、RedoclyとOpenAPI仕様を使用して美しいAPIドキュメントを生成する方法を説明します。

機能

  • OpenAPI 3.0仕様によるAPI定義
  • Redoclyによるドキュメント生成
  • Express.jsサーバーと統合されたドキュメント
  • ボリュームマウントを使用したDocker Compose設定

クイックスタート

Docker Composeを使用（推奨）

# APIサーバーとドキュメントサーバーを起動
docker-compose up

# APIサーバーのみ起動
docker-compose up api

# ドキュメントサーバーのみ起動
docker-compose up docs

ドキュメントへのアクセス: http://localhost:8080

ローカル開発

# 依存関係をインストール
npm install

# APIサーバーを起動
npm run dev

# ドキュメントを生成
npm run docs:build

# ドキュメントをプレビュー
npm run docs:serve

# OpenAPI仕様をリント
npm run docs:lint

エンドポイント

エンドポイント説明
http://localhost:3000/api/v1/usersユーザーAPI
http://localhost:3000/api/v1/products商品API
http://localhost:3000/api-docsRedocドキュメント
http://localhost:3000/openapi.yamlOpenAPI仕様ファイル
http://localhost:8080/redoc-static.html静的ドキュメント（ビルド後）

プロジェクト構造

redocly-openapi-example/
├── api/
│   └── openapi.yaml      # OpenAPI仕様
├── src/
│   └── index.ts          # Expressサーバー
├── docs/                  # 生成されたドキュメント
├── docker-compose.yml
├── package.json
├── tsconfig.json
└── redocly.yaml          # Redocly設定

コマンド説明

npm run docs:build静的HTMLドキュメントを生成
