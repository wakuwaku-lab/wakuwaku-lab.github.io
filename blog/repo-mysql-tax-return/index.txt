# MySQLで確定申告

このプロジェクトは、MySQLを使用した複式簿記による青色申告の帳簿作成例を提供します。

## 概要

 MySQLを使って青色申告所需的以下の帳簿を作成します：

- 仕訳帳
- 総勘定元帳
- 試算表
- 精算表
- 損益計算書
- 貸借対照表

## システム構成

```mermaid
flowchart LR
    A[仕訳帳.csv] --> B[MySQL]
    C[勘定科目マスタ.csv] --> B
    B --> D[Views]
    D --> E[各種帳簿出力]
    
    subgraph DB["データベース"]
    B
    D
    end
    
    subgraph Views["ビュー"]
    D1[総勘定元帳]
    D2[試算表]
    D3[精算表]
    D4[損益計算書]
    D5[貸借対照表]
    end
```

## 複式簿記のフロー

```mermaid
flowchart TD
    A[仕訳帳] -->|転記| B[総勘定元帳]
    B -->|集計| C[試算表]
    C -->|決算整理| D[精算表]
    D --> E[損益計算書]
    D --> F[貸借対照表]
```

## 必要なもの

- Docker
- Docker Compose

## 始め方

1. Docker ComposeでMySQLを起動：

```bash
cd mysql-tax-return
docker-compose up -d
```

2. データのインポートが完了するまで待つ（自動的に行われます）

3. MySQLに接続して確認：

```bash
docker exec -it mysql-tax-return mysql -u root -prootpassword --default-character-set=utf8mb4 tax_accounting
```

## 帳簿の確認

### 試算表

```sql
SELECT * FROM 試算表_印刷用;
```

### 損益計算書

```sql
SELECT * FROM 損益計算書;
```

### 貸借対照表

```sql
SELECT * FROM 貸借対照表;
```

### 精算表

```sql
SELECT * FROM 精算表_印刷用;
```

### 総勘定元帳（勘定科目別）

```sql
-- 売上の総勘定元帳
SELECT * FROM 総勘定元帳_売上_印刷用;

-- 現金の総勘定元帳
SELECT * FROM 総勘定元帳_現金_印刷用;
```

## テーブル構造

### 仕訳帳

| カラム | 型 | 説明 |
|--------|-----|------|
| id | INT | 主キー |
| 日付 | DATE | 取引日 |
| 借方勘定科目 | VARCHAR(8) | 借方勘定科目 |
| 借方金額 | DECIMAL(10,0) | 借方金額 |
| 貸方勘定科目 | VARCHAR(8) | 貸方勘定科目 |
| 貸方金額 | DECIMAL(10,0) | 貸方金額 |
| 摘要 | VARCHAR(255) | 摘要 |
| 備考 | VARCHAR(8) | 備考（決算整理フラグ） |

### 勘定科目マスタ

| カラム | 型 | 説明 |
|--------|-----|------|
| id | INT | 主キー |
| 科目名 | VARCHAR(8) | 勘定科目名 |
| 貸借区分 | VARCHAR(2) | 借方/貸方 |
| 精算表 | VARCHAR(5) | 貸借対照表/損益計算書 |

## 青色申告特別控除

複式簿記により記帳することで、 最大65万円の青色申告特別控除を受けることができます。

## 参考

- [MySQLで確定申告 - KRAY Inc.](https://kray.jp/blog/file-a-tax-return-with-mysql/)
- [国税庁HP 青色申告制度](https://www.nta.go.jp/taxanswer/shotoku/2070.htm)
