# htmx Task Manager Demo

htmx を使用した動的UIの実装例です。JavaScript をほとんど書かずに、HTML 属性だけで動的なインタラクションを実現できます。

## 機能

- **インクリメンタル検索**: 検索ボックスに入力するとリアルタイムでタスクが絞り込まれます（300ms デバウンス付き）
- **インライン編集**: タスクのタイトルやステータスをクリックして直接編集
- **削除アニメーション**: 確認ダイアログ付きの削除、フェードアウトアニメーション
- **詳細パネル**: タスクを選択して詳細情報を表示

## 使い方

`src/index.html` をブラウザで開くだけで動作します。

```bash
# ローカルサーバーで開く場合
cd htmx-task-manager/src
python -m http.server 8000
# または
npx serve
```

ブラウザで `http://localhost:8000` にアクセス

## htmx の主要属性

| 属性 | 説明 |
|------|------|
| `hx-get` | GET リクエストを送信 |
| `hx-post` | POST リクエストを送信 |
| `hx-patch` | PATCH リクエストを送信 |
| `hx-delete` | DELETE リクエストを送信 |
| `hx-target` | レスポンスを挿入する先を指定 |
| `hx-trigger` | リクエストが発生するトリガーを指定 |
| `hx-swap` | レスポンスの挿入方法を指定 |
| `hx-indicator` | ローディング表示用の要素を指定 |
| `hx-confirm` | 確認ダイアログを表示 |

## 参考

- [htmx 公式サイト](https://htmx.org/)
- [htmx Examples](https://htmx.org/examples/)
- [HTMLを拡張し、JSなしで動的UIを作る「htmx」 | gihyo.jp](https://gihyo.jp/article/2026/02/htmx-introduction)
