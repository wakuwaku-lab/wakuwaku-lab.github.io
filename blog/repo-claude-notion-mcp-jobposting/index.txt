# Claude Code × Notion MCP 求人票作成デモ

このプロジェクトは、Claude CodeとNotion MCPを組み合わせた業務自動化のデモです。Findy Tech Blogの記事「1時間を5分に短縮！Claude Code × Notion MCPで実現した業務自動化の全貌」に基づいて作成されました。

## 概要

このデモは、企業名と既存求人のURLを入力すると、Notionから企業情報を自動取得し、求人票ドラフトを生成するワークフローを再現しています。

### 主な機能

- Notionからの企業情報自動取得
- 既存求人情報の取得
- 求人票ドラフトの自動生成
- 確認が必要な項目の明確化

## 必要環境

- Node.js 18以上
- npm

## インストール

```bash
npm install
```

## 実行方法

### インタラクティブモード

```bash
npm start
```

### デモモード（自動実行）

```bash
npm run demo
```

## 出力例

```
===========================================
   求人票作成アシスタント v1.0 (Demo Mode)
===========================================

求人票作成を開始します。

企業名を入力してください:
> ABC株式会社

既存求人のURLを入力してください:
> https://ai-career.findy-code.io/companies/abc/jobs/001

🔍 Notionから ABC株式会社 の情報を検索します...

✅ 求人票ドラフトを作成しました！
📁 保存先: src/drafts/ABC_____jobposting_xxx.md

【生成された求人票】

# 求人票: ABC株式会社

## 基本情報
- 職種: AIエンジニア
- 年収: 年収 600万円〜1000万円

## 技術スタック
- Python
- React
- AWS
- TensorFlow

## 確認が必要な項目
- 生成AI活用ラベル: 4つの選択肢から確認が必要
- AI活用に対する評価制度: 既存求人に記載なし
```

## プロジェクト構造

```
claude-notion-mcp-jobposting/
├── .claude/
│   ├── CLAUDE.md          # カスタムスラッシュコマンド定義
│   └── claude.json        # MCP設定
├── src/
│   ├── index.js           # メインスクリプト
│   ├── data/
│   │   └── companies.json # 企業データ
│   ├── drafts/            # 生成された求人票
│   └── setup.sh           # セットアップスクリプト
├── docs/                  # スライド資料
├── package.json
└── README.md
```

## アーキテクチャ

### カスタムスラッシュコマンド

`.claude/CLAUDE.md`に定義されたカスタムスラッシュコマンド `/create-job-posting` を使用することで、Claude Code上で対話的に求人票を作成できます。

### MCP（Model Context Protocol）

Notion MCPを使用することで、Claude CodeがNotionのデータベースにアクセスし、企業情報を自動取得します。

### プロンプト設計の工夫

- **「雰囲気で書かない」原則**: 情報がない項目は推測で埋めず、「確認が必要な項目」としてリストアップ
- **既存求人からの情報取得**: API経由で正確にコピー
- **暗黙知のコード化**: メンバーごとに異なる判断基準をプロンプトに明記

## 実際のClaude Codeでの使用

1. Claude Codeをインストール
2. Notion APIキーを取得・設定
3. MCPを設定
4. `/create-job-posting`コマンドを実行

詳細については [Findy Tech Blog](https://tech.findy.co.jp/entry/2026/03/17/070000) を参照してください。

# Claude Code × Notion MCP 招聘单创建演示

此项目是结合Claude Code和Notion MCP实现业务自动化的演示。参考Findy Tech Blog的文章「1時間を5分に短縮！Claude Code × Notion MCPで実現した業務自動化の全貌」创建。

此演示重现了输入企业名称和现有招聘信息的URL后，从Notion自动获取企业信息并生成招聘单草案的工作流程。

### 主要功能

- 从Notion自动获取企业信息
- 获取现有招聘信息
- 自动生成招聘单草案
- 明确标注需要确认的项目

## 环境要求

- Node.js 18以上
- npm

npm install

## 架构

### 自定义斜杠命令

使用`.claude/CLAUDE.md`中定义的自定义斜杠命令 `/create-job-posting`，可以在Claude Code上通过对话创建招聘单。

### MCP（Model Context Protocol）

通过使用Notion MCP，Claude Code可以访问Notion的数据库，自动获取企业信息。

### 提示词设计要点

- **「不凭感觉写」原则**: 没有信息的项目不推测填写，而是列为「需要确认的项目」
- **从现有招聘信息获取**: 通过API准确复制
- **隐性知识的代码化**: 将因人而异的判断标准明确写入提示词

1. 安装Claude Code

# Claude Code × Notion MCP Job Posting Demo

This project is a demo of business automation combining Claude Code and Notion MCP. Created based on the Findy Tech Blog article "1時間を5分に短縮！Claude Code × Notion MCPで実現した業務自動化の全貌".

## Overview

This demo recreates the workflow of entering a company name and existing job posting URL, automatically fetching company information from Notion, and generating a job posting draft.

### Key Features

- Automatic company information retrieval from Notion
- Fetch existing job posting information
- Automatic job posting draft generation
- Clear identification of items requiring confirmation

## Requirements

- Node.js 18+
- npm

## Installation

```bash
npm install
```

## Usage

### Interactive Mode

```bash
npm start
```

### Demo Mode (Automatic)

```bash
npm run demo
```

## Example Output

```
===========================================
   Job Posting Assistant v1.0 (Demo Mode)
===========================================

Starting job posting creation.

Enter company name:
> ABC株式会社

Enter existing job posting URL:
> https://ai-career.findy-code.io/companies/abc/jobs/001

🔍 Searching Notion for ABC株式会社...

✅ Job posting draft created!
📁 Saved to: src/drafts/ABC_____jobposting_xxx.md

【Generated Job Posting】

# Job Posting: ABC株式会社

## Basic Information
- Position: AI Engineer
- Salary: ¥6,000,000 - ¥10,000,000

## Tech Stack
- Python
- React
- AWS
- TensorFlow

## Items Requiring Confirmation
- Generative AI usage label: Needs confirmation from 4 options
- AI usage evaluation system: Not in existing posting
```

## Project Structure

```
claude-notion-mcp-jobposting/
├── .claude/
│   ├── CLAUDE.md          # Custom slash command definition
│   └── claude.json        # MCP configuration
├── src/
│   ├── index.js           # Main script
│   ├── data/
│   │   └── companies.json # Company data
│   ├── drafts/            # Generated job postings
│   └── setup.sh           # Setup script
├── docs/                  # Slide materials
├── package.json
└── README.md
```

## Architecture

### Custom Slash Command

Using the custom slash command `/create-job-posting` defined in `.claude/CLAUDE.md`, you can interactively create job postings within Claude Code.

### MCP (Model Context Protocol)

Using Notion MCP, Claude Code can access Notion databases and automatically retrieve company information.

### Prompt Design Principles

- **"Don't guess" principle**: Items without information are not filled in with guesses, but listed as "Items Requiring Confirmation"
- **Information from existing postings**: Accurately copied via API
- **Codifying tacit knowledge**: Explicitly stating judgment criteria that varied between team members

## Using with Actual Claude Code

1. Install Claude Code
2. Get and set Notion API key
3. Configure MCP
4. Execute `/create-job-posting` command

See [Findy Tech Blog](https://tech.findy.co.jp/entry/2026/03/17/070000) for details.
