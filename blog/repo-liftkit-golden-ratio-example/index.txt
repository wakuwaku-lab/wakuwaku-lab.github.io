# LiftKit Golden Ratio Example

黄金比（φ ≈ 1.618）に基づいてUIを自動生成するNext.js UIフレームワーク「LiftKit」のサンプルプロジェクトです。

## 概要

LiftKitは、「完璧主義者のためのNext.js UIフレームワーク」として知られるオープンソースのUIフレームワークです。黄金比（φ = 1.618...）に基づいて、すべての余白・フォントサイズ・ボーダー半径を自動計算し、「なぜか心地よいUI」を提供します。

### 主な特徴

- **黄金比ベースのスペーシング**: すべてのサイズと間隔が黄金比から導出される
- **Dynamic Color**: リアルタイムにカラーテーマをプレビュー・変更可能
- **豊富なコンポーネント**: Badge、Button、Card、Inputなどの組み込みコンポーネント
- **Apache-2.0ライセンス**: 商利用可能な無料オープンソース

## 必要環境

- Node.js 18.x以上
- npm 9.x以上
- Git

## インストール方法

### 手順1: 依存関係をインストール

```bash
cd liftkit-golden-ratio-example
npm install
```

### 手順2: 開発サーバーを起動

```bash
npm run dev
```

ブラウザで http://localhost:3000 にアクセスすると、サンプルページが表示されます。

## プロジェクトの構造

```
liftkit-golden-ratio-example/
├── src/
│   ├── app/
│   │   ├── globals.css    # LiftKit CSS変数とスタイル
│   │   ├── layout.tsx     # ルートレイアウト
│   │   └── page.tsx       # メインページ（ демо）
│   └── lib/css/
│       └── index.css      # LiftKit Core CSS
├── package.json
├── tsconfig.json
└── next.config.ts
```

## デモ内容

このサンプルプロジェクトには以下のデモが含まれています：

1. **Dynamic Color（動的テーマ変更）**: 5つのカラーテーマ（Default、Ocean、Forest、Sunset、Royal）をリアルタイムに切り替え可能
2. **黄金比スペーシング**: φ⁻³〜φ⁰のスペーシング例を表示
3. **UIコンポーネント**: Card、Button、Inputなどのサンプル
4. **タイポグラフィスケール**: 黄金比ベースのフォントサイズ例

## 黄金比について

**黄金比（φ = 1.618...）** は、古代ギリシャから「最も美しい比率」として知られる数学定数です。ひまわりの種の配列、貝殻のらせん、パルテノン神殿など自然界や歴史的建造物の至るところに現れ、人間が直感的に「美しい」と感じる比率とされています。

LiftKitはこの比率をUIのスペーシング・タイポグラフィ・ボーダー半径すべてに適用し、開発者が細かなピクセル調整コードを書かずとも、視覚的に洗練されたUIを生み出します。

## 次のステップ

実際のプロジェクトでLiftKitを使用するには：

1. **新規プロジェクトの場合**:
   ```bash
   git clone https://github.com/Chainlift/liftkit-template.git
   cd liftkit-template
   npm install
   npm run dev
   ```

2. **既存のNext.jsプロジェクトに追加する場合**:
   ```bash
   npm install @chainlift/liftkit --save-dev
   npx liftkit init
   ```

詳細な情報は以下のリンクを参照してください：

- [GitHub - Chainlift/liftkit](https://github.com/Chainlift/liftkit)
- [LiftKit インストールドキュメント](https://www.chainlift.io/liftkit/install)

# LiftKit Golden Ratio Example

This is a sample project for LiftKit, a Next.js UI framework that automatically generates UIs based on the golden ratio (φ ≈ 1.618).

## Overview

LiftKit is an open-source UI framework known as "the Next.js UI framework for perfectionists." It automatically calculates all spacing, font sizes, and border radii based on the golden ratio (φ = 1.618...), providing UIs that feel inherently pleasing.

### Key Features

- **Golden Ratio Spacing**: All sizes and spacing derived from the golden ratio
- **Dynamic Color**: Preview and change color themes in real-time
- **Rich Components**: Built-in components like Badge, Button, Card, Input
- **Apache-2.0 License**: Free open-source software for commercial use

## Requirements

- Node.js 18.x or higher
- npm 9.x or higher
- Git

## Installation

### Step 1: Install dependencies

```bash
cd liftkit-golden-ratio-example
npm install
```

### Step 2: Start the development server

```bash
npm run dev
```

Open http://localhost:3000 in your browser to view the sample page.

## Project Structure

```
liftkit-golden-ratio-example/
├── src/
│   ├── app/
│   │   ├── globals.css    # LiftKit CSS variables and styles
│   │   ├── layout.tsx     # Root layout
│   │   └── page.tsx       # Main page (demo)
│   └── lib/css/
│       └── index.css      # LiftKit Core CSS
├── package.json
├── tsconfig.json
└── next.config.ts
```

## Demo Contents

This sample project includes the following demos:

1. **Dynamic Color**: 5 color themes (Default, Ocean, Forest, Sunset, Royal) with real-time switching
2. **Golden Ratio Spacing**: Examples of φ⁻³〜φ⁰ spacing
3. **UI Components**: Samples of Card, Button, Input, etc.
4. **Typography Scale**: Golden ratio-based font size examples

## About the Golden Ratio

**The golden ratio (φ = 1.618...)** is a mathematical constant known since ancient Greece as "the most beautiful ratio." It appears throughout nature and historic architecture—in sunflower seed arrangements, shell spirals, the Parthenon—and is considered a ratio that humans intuitively perceive as "beautiful."

LiftKit applies this ratio to all UI spacing, typography, and border radii, enabling developers to create visually refined UIs without writing detailed pixel-adjustment code.

   ```bash
   npm install
   npm run dev
   ```

2. **To add to an existing Next.js project**:
   ```bash
   npm install @chainlift/liftkit --save-dev
   npx liftkit init
   ```
