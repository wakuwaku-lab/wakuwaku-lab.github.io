# station-property-search

最寄り駅から指定時間内で到達可能な物件を地図上に表示するアプリケーション

## 概要

このアプリケーションは、最寄り駅と交通手段（徒歩・自転車・車）、所要時間を指定することで、その条件に当てはまる物件を地図上にピンを刺して表示します。

元記事: [【個人開発】最寄り駅から徒歩X分以内の物件を地図上に列挙できるアプリを作ってみた](https://qiita.com/nu197ma/items/bc9aee9c5951e01ad13b)

## 機能

- 駅名入力（自動補完）
- 交通手段選択（徒歩・自転車・車）
- 所要時間設定（1-60分）
- 地図上的到達可能範囲の表示
- 物件マーカーとルート表示
- 物件リスト表示

## 技術スタック

- React 18 + TypeScript
- Vite
- Leaflet (react-leaflet)
- OSRM Routing API

##  설치

```bash
cd station-property-search
npm install
```

## 실행

```bash
npm run dev
```

ブラウザで http://localhost:5173 を開きます。

## 使用方法

1. 最寄りの駅名を入力してください（例：新宿駅、渋谷駅）
2. 交通手段を選択してください（徒歩・自転車・車）
3. 所要時間を入力してください（分）
4. 「検索」ボタンをクリックすると、条件に一致する物件が地図とリストに表示されます
5. 物件をクリックすると、駅からのルートが表示されます

## デモ

![Demo](https://example.com/demo.png)

## アーキテクチャ

```
src/
├── main.tsx       # エントリーポイント
├── App.tsx        # メインコンポーネント
├── index.css      # スタイル
└── vite-env.d.ts  # 型定義
```

# station-property-search

A web application that displays properties reachable within a specified time from the nearest station on a map.

## Overview

This application allows users to search for rental properties by specifying:
- Nearest station name
- Transportation mode (walk, bicycle, car)
- Travel time (in minutes)

Properties within the specified criteria are displayed as markers on the map with route visualization.

Based on: [【個人開発】最寄り？谢邀X分以内の物件を地図上に列挙できるアプリを作ってみた](https://qiita.com/nu197ma/items/bc9aee9c5951e01ad13b)

## Features

- Station name input with autocomplete
- Transportation mode selection (walk/bicycle/car)
- Configurable travel time (1-60 minutes)
- Reachable area visualization on map
- Property markers with route display
- Property list panel

## Tech Stack

- React 18 + TypeScript
- Vite
- Leaflet (react-leaflet)
- OSRM Routing API

## Installation

```bash
cd station-property-search
npm install
```

## Running

```bash
npm run dev
```

Open http://localhost:5173 in your browser.

## Usage

1. Enter a station name (e.g., Shinjuku, Shibuya)
2. Select transportation mode
3. Set travel time in minutes
4. Click "Search" to find properties
5. Click on a property to see the route from the station

## Architecture

```
src/
├── main.tsx       # Entry point
├── App.tsx        # Main component
├── index.css      # Styles
└── vite-env.d.ts  # Type definitions
```
