Remotion 入門ガイド

Remotionを使用した動画作成のサンプルプロジェクトです。

Remotionとは

Remotionは、Reactを使ってプログラムで動画を作成できるツールです。パラメータによる動画生成、テンプレートベースの大量生産、サーバーサイドレンダリングワークフローなどが可能です。

前提条件

  • Node.js (v18以上)
  • npm または yarn

インストール

cd remotion-beginner-guide
npm install

使い方

開発プレビュー

開発サーバーを起動して動画をプレビューします：

npm run dev

MP4にレンダリング

MP4ファイルとして動画をエクスポートします：

npm run build

または、Remotion CLIを直接使用します：

# 日本語バージョン
npx remotion render HelloWorld out/video-ja.mp4

# 英語バージョン
npx remotion render HelloWorldEn out/video-en.mp4

# 中国語バージョン
npx remotion render HelloWorldZh out/video-zh.mp4

出力は out/ ディレクトリに保存されます。

i18n（多言語対応）

このプロジェクトは3つの言語をサポートしています：

  • 日本語 (ja) - デフォルト
  • 英語 (en)
  • 中国語 (zh)

言語ファイルは src/i18n/ ディレクトリにあります：

src/i18n/
├── ja.json  # 日本語（デフォルト）
├── en.json  # 英語
└── zh.json  # 中国語

コンポーネントでi18nを使用

import { getTranslations, Language } from "./i18n";

interface Props {
  lang?: Language;
}

export const MyComponent: React.FC<Props> = ({ lang = "ja" }) => {
  const t = getTranslations(lang);
  return <h1>{t.title}</h1>;
};

プロジェクト構造

remotion-beginner-guide/
├── src/
│   ├── HelloRemotion.tsx   # メイン動画コンポーネント
│   ├── root.tsx            # エントリーポイント
│   └── i18n/
│       ├── index.ts        # i18nユーティリティ
│       ├── ja.json        # 日本語翻訳
│       ├── en.json        # 英語翻訳
│       └── zh.json        # 中国語翻訳
├── package.json
├── tsconfig.json
└── remotion.config.ts     # Remotion設定

主なコンセプト

  • useCurrentFrame(): 動画内の現在のフレーム番号を取得
  • interpolate(): ある範囲から別の範囲に値をマッピング（例：フレームから不透明度へ）
  • spring(): バネ物理ベースのアニメーション
