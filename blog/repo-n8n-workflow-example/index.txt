# n8n-workflow-example

n8n（ワークフロー自動化ツール）の示例プロジェクトです。

## 概要

このプロジェクトは、n8nを使ったワークフロー自動化の示例を提供します。

## 必要な環境

- Docker
- Docker Compose

## クイックスタート

```bash
cd src
./setup.sh
```

または、手動で起動する場合：

```bash
cd src
docker-compose up -d
```

## アクセス

- **URL**: http://localhost:5678
- **ユーザー名**: admin
- **パスワード**: admin123

## サンプルワークフロー

### 1. Hello World Workflow
最も基本的なワークフローです。手動でトリガーし、メッセージを出力します。

**ファイル**: `workflows/hello-world.json`

### 2. Schedule Workflow
5分ごとに定期実行されるワークフローです。HTTPリクエストを実行し、データを処理します。

**ファイル**: `workflows/schedule-workflow.json`

### 3. Webhook Workflow
Webhookを受け取り、データを処理してレスポンスを返すワークフローです。

**ファイル**: `workflows/webhook-workflow.json`

## ワークフローのインポート方法

1. n8nの左メニューから「Workflows」をクリック
2. 「Import from File」をクリック
3. `workflows/`フォルダ内のJSONファイルを選択
4. 「Import」をクリック

## ワークフローの実行

1. ワークフローを開く
2. 「Execute Workflow」をクリック
3. 結果を確認

## Docker コマンド

```bash
# 起動
docker-compose up -d

# 停止
docker-compose down

# ログ確認
docker-compose logs -f

# 再起動
docker-compose restart
```

## プロジェクト構造

```
n8n-workflow-example/
├── src/
│   ├── docker-compose.yml    # Docker設定
│   ├── setup.sh              # セットアップスクリプト
│   └── workflows/            # サンプルワークフロー
│       ├── hello-world.json
│       ├── schedule-workflow.json
│       └── webhook-workflow.json
└── README.md
```

# n8n-workflow-example

Example project for n8n (workflow automation tool).

## Overview

This project provides examples of workflow automation using n8n.

## Requirements

- Docker
- Docker Compose

## Quick Start

```bash
cd src
./setup.sh
```

Or manually:

```bash
cd src
docker-compose up -d
```

## Access

- **URL**: http://localhost:5678
- **Username**: admin
- **Password**: admin123

## Sample Workflows

### 1. Hello World Workflow
The most basic workflow. Manually triggered to output a message.

**File**: `workflows/hello-world.json`

### 2. Schedule Workflow
A workflow that runs periodically every 5 minutes. Makes HTTP requests and processes data.

**File**: `workflows/schedule-workflow.json`

### 3. Webhook Workflow
A workflow that receives webhooks, processes data, and returns a response.

**File**: `workflows/webhook-workflow.json`

## How to Import Workflows

1. Click "Workflows" in the n8n left menu
2. Click "Import from File"
3. Select a JSON file from the `workflows/` folder
4. Click "Import"

## Running a Workflow

1. Open the workflow
2. Click "Execute Workflow"
3. View the results

## Docker Commands

```bash
# Start
docker-compose up -d

# Stop
docker-compose down

# View logs
docker-compose logs -f

# Restart
docker-compose restart
```
