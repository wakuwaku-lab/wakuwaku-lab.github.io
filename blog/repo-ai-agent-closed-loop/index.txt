# AI Agent Closed Loop

AIエージェントのクローズドループシステムを演示するプロジェクトです。OpenClawのコンセプトに基づいています。

## 概要

このプロジェクトは、AIエージェントがフィードバックを収集し、自身の行動を自動的に調整する「クローズドループ」アーキテクチャの実装例です。

### 主な機能

- **ペルソナ設定**: SOUL.md でAIの性格、ルール、原則を定義
- **メモリシステム**: 長期記憶と短期記憶の管理
- **クローズドループフィードバック**: 実行結果から学習し、改善
- **マルチチャネル対応**: Discord、Telegramに対応
- **ツール統合**: Web検索、ファイル操作、コマンド実行等功能

## 必要環境

- Node.js 18.0.0以上

## インストール

```bash
cd ai-agent-closed-loop
npm install
```

## 実行

```bash
npm start
```

## 設定

### SOUL.md (ペルソナ設定)

AIのエージェントの性格と行動原則を定義します：

```markdown
# クロウ (Agent Name)

## 性格
- 礼儀正しい
- 前向き

## ルール
- 用户的喜好を考慮する

## 原則
- 秘密情報を漏らさない
```

### USER.md (ユーザー設定)

ユーザーごとの趣向やコンテキストを設定できます。

### 環境変数

```bash
DISCORD_TOKEN=your_discord_token
TELEGRAM_TOKEN=your_telegram_token
```

## 使用例

```
=== AI Agent Closed Loop System ===

Agent: クロウ
Model: claude-sonnet-4

--- Test 1: シンプルクエリ ---
[クロウ] Planning: こんにちは...
Result: 処理が完了しました。

--- Test 2: 検索クエリ ---
[クロウ] Executing with tools: webSearch
Result: 1/1 の操作が完了しました。

--- Test 3: クローズドループフィードバック ---
[クロウ] Adjusting based on feedback...
```

## アーキテクチャ

```
┌─────────────┐     ┌──────────────┐     ┌─────────────┐
│  Input      │────▶│  Planning    │────▶│  Execution  │
│  (User)     │     │  (Intent)    │     │  (Tools)    │
└─────────────┘     └──────────────┘     └─────────────┘
                                               │
                    ┌──────────────┐           │
                    │  Adjustment  │◀──────────┘
                    │  (Feedback)  │
                    └──────────────┘
                           │
                    ┌──────────────┐
                    │   Memory     │
                    │  (Storage)   │
                    └──────────────┘
```

### コンポーネント

1. **ClosedLoopAgent**: メインのエージェントクラス
2. **AgentMemory**: メモリ管理システム
3. **ToolRegistry**: ツール登録・実行
4. **FeedbackLoop**: フィードバック収集・分析
5. **ChannelManager**: マルチチャネル管理

# AI Agent Closed Loop

A project demonstrating an AI agent closed-loop system. Based on OpenClaw concepts.

## Overview

This project implements an example of "closed-loop" architecture where AI agents learn from feedback and automatically adjust their behavior.

### Key Features

- **Persona Configuration**: Define AI personality, rules, and principles in SOUL.md
- **Memory System**: Long-term and short-term memory management
- **Closed-Loop Feedback**: Learn from execution results and improve
- **Multi-Channel Support**: Discord, Telegram integration
- **Tool Integration**: Web search, file operations, command execution

## Requirements

- Node.js 18.0.0 or higher

## Installation

```bash
cd ai-agent-closed-loop
npm install
```

## Run

```bash
npm start
```

## Configuration

### SOUL.md (Persona Configuration)

Define the AI agent's personality and behavioral principles:

```markdown
# Crow (Agent Name)

## Personality
- Polite
- Positive

## Rules
- Consider user preferences

## Principles
- Never leak confidential information
```

### USER.md (User Settings)

Set user preferences and context.

### Environment Variables

```bash
DISCORD_TOKEN=your_discord_token
TELEGRAM_TOKEN=your_telegram_token
```

## Example Output

```
=== AI Agent Closed Loop System ===

Agent: Crow
Model: claude-sonnet-4

--- Test 1: Simple Query ---
[Crow] Planning: Hello...
Result: Processing completed.

--- Test 2: Search Query ---
[Crow] Executing with tools: webSearch
Result: 1/1 operations completed.

--- Test 3: Closed Loop Feedback ---
[Crow] Adjusting based on feedback...
```
