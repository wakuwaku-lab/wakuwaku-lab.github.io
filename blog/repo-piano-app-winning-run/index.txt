# Piano Score Player

An interactive piano score player built with React. Renders MusicXML sheet music and plays it back with real piano samples.

## Features

- 🎼 Sheet music rendering via [OpenSheetMusicDisplay](https://opensheetmusicdisplay.org/)
- 🎹 Real acoustic grand piano samples (MusyngKite soundfont)
- ▶️ Playback with cursor tracking synchronized to audio
- ⌨️ Keyboard input to play notes manually
- 🖱️ Click piano keys to play
- 🎚️ Adjustable tempo (BPM)

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Controls

| Control | Action |
|---------|--------|
| Space | Play / Stop |
| ▶ Play button | Play / Stop |
| ⏮ Reset button | Return to beginning |
| Tempo slider | Adjust playback speed |
| Piano keys (click) | Play individual notes |
| Keyboard `a s d f g h j k l` | Play notes C4–E5 |
| Keyboard `w e t y u o p` | Play black keys |

## Tech Stack

- React + Vite
- [opensheetmusicdisplay](https://github.com/opensheetmusicdisplay/opensheetmusicdisplay)
- [soundfont-player](https://github.com/danigb/soundfont-player)
- Web Audio API
