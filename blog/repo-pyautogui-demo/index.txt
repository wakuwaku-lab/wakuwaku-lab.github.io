# PyAutoGUI Demo

A comprehensive demonstration of PyAutoGUI - Python library for GUI automation.

## Features Demonstrated

- **Mouse Operations**: Move, click, drag, scroll
- **Keyboard Operations**: Text input, special keys, hotkeys
- **Screen Operations**: Screenshot capture, pixel color detection
- **Message Boxes**: Alert, confirm, prompt, password dialogs
- **Safety Features**: FAILSAFE, PAUSE settings
- **Tween/Easing**: Smooth mouse movement animations

## Installation

```bash
pip install -r requirements.txt
```

### Linux Dependencies

```bash
sudo apt-get install python3-xlib scrot python3-tk python3-dev
```

### macOS

Grant accessibility permissions in System Preferences > Privacy & Security > Accessibility

## Usage

Run full demo:
```bash
python src/main.py
```

Run specific demo modes:
```bash
python src/main.py mouse     # Mouse operations only
python src/main.py keyboard  # Keyboard operations only
python src/main.py screen    # Screen operations only
python src/main.py safety    # Safety features only
```

## Demo Modules

| Module | Description |
|--------|-------------|
| `demo_mouse_movement()` | Move cursor to coordinates with optional duration |
| `demo_mouse_click()` | Left, right, double clicks |
| `demo_mouse_drag()` | Drag with left button held |
| `demo_mouse_scroll()` | Scroll up/down |
| `demo_keyboard_write()` | Type text strings |
| `demo_keyboard_press()` | Press individual keys |
| `demo_hotkey()` | Keyboard shortcuts (Ctrl+C, etc.) |
| `demo_screenshot()` | Capture screen images |
| `demo_pixel_info()` | Get pixel colors |
| `demo_message_boxes()` | Show dialogs |
| `demo_safety_features()` | FAILSAFE and PAUSE |
| `demo_tween_easing()` | Smooth animations |

## Safety Notes

- **FAILSAFE**: Move mouse to top-left corner (0,0) to stop execution
- **PAUSE**: Add delay between operations to prevent runaway scripts
- Always test in a safe environment first
